## рҹҺҜ Overview

This document outlines the user flows for StreamHub Dashboard Management System across three roles:

- **USER**: View-only access to assigned dashboards
- **MODERATOR**: Create and manage folders, create/edit dashboards in assigned folders
- **ADMIN**: Full system access including company, user, and permission management

**Key Principles:**
- All users must authenticate via Google OAuth
- Access is permission-based (not company-based for resources)
- Admins have global cross-company access
- Users see dashboards based on **3-layer permission model** (see [Roles & Permissions Guide > Permission Structure](../GUIDES/roles-and-permissions.md#-permission-structure))
- Moderators manage folders via `assignedFolders` array
- **Moderators have 2 views:** Viewer mode (like User) and Manager mode (like limited Admin) вҖ” see [Moderator Dual-View Model](../GUIDES/roles-and-permissions.md#-moderator-dual-view-model)
- **Tag system** provides cross-cutting dashboard categorization вҖ” Folder = physical location, Tag = logical grouping
- All roles can **filter dashboards by tags** via the "View All" page
- Sidebar navigation is **role-based** вҖ” see [Sidebar Navigation Wireframe](wireframes/sidebar-navigation.md)

**рҹ“Ң Note:** For complete permission structure details, access logic, and Firestore security rules, refer to [Roles & Permissions Guide](../GUIDES/roles-and-permissions.md) - the single source of truth for all role and permission definitions.

### USER Flow Decision Points

| Decision Point | Outcome | Next Step |
|---|---|---|
| **Has Google Account?** | вң… Yes | Authenticate and load profile |
| | вқҢ No | Show Google Sign-In button |
| **Has View Permission?** | вң… Yes | Display dashboard in list |
| | вқҢ No | Hide dashboard from list |
| **Tag filter selected?** | вң… Yes | Show only dashboards with matching tags |
| | вқҢ No (All) | Show all accessible dashboards |
| **Folder filter selected?** | вң… Yes | Show only dashboards in selected folder |
| | вқҢ No (All) | Show dashboards from all folders |
| **More dashboards to load?** | вң… Yes | Lazy load next 12 items on scroll |
| | вқҢ No | Show "all loaded" indicator |
| **Expired Session?** | вң… Yes | Refresh token / Re-authenticate |
| | вқҢ No | Continue using app |
