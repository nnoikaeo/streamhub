# Firestore Collections Reference

Quick reference for all Firestore collections.

(See [Database Schema](../../GUIDES/database-schema.md) for detailed information)

## Collections

| Collection | Purpose | Documents |
|---|---|---|
| `users/` | User profiles | `{userId}` |
| `dashboards/` | Dashboard configs | `{dashboardId}` |
| `documents/` | File metadata | `{documentId}` |
| `settings/` | App configuration | `{settingId}` |
